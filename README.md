# 🎮 Tic-Tac-Toe (Mario vs Luigi)

A 2-player Tic-Tac-Toe game built using Godot.

## 🚀 Live Demo
[Play Here](PASTE_ITCH_LINK)

---

## 🎯 Features
- Mario vs Luigi themed gameplay
- Sound effects and background music
- Menu system (Main, Options)
- Playable in browser

---

## 🕹 Controls
- Click → Place marker
- Ctrl + R → Reset 

---

## 🧠 Game Logic
- Players alternate turns
- First to get 3 in a row wins
- Full board with no winner → Draw

---

## 🧪 Testing
Unit tests implemented using GUT framework.

To run tests:
- Open GUT panel in Godot
- Click "Run All Tests"

---

## 🤖 AI Usage
AI tools were used for:
